public long getComponentId (int position)

Return the stable ID for the component at position. If hasStableIds() would return false this method should return RecyclerView. The default implementation of this method returns RecyclerView.

Parameters
position Binder position to query
Returns
  • the stable ID of the component at position

public final boolean hasStableIds ()

Returns true if this binder publishes a unique long value that can act as a key for the component at a given position in the data set. If that component is relocated in the data set, the ID returned for that component should be the same.

Returns
  • true if this binder's component have stable IDs
