public void notifyItemRangeRemoved (int positionStart, int itemCount)

Notifies this Binder that a range of items were removed at index position. The items at index position + itemCount + i (with i >= 0) before notifyRangeRemoved was called will have index position + i after calling this method.

public void notifyItemRemoved (int position)

Notifies this Binder that an item was removed from index position. The items at index position + i (with i >= 1) before notifyItemRemoved was called will have index position + i - 1 after calling this method.
