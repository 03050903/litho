public abstract Component<?> createComponent (ComponentContext c, int position)

Create a Component for the given position.

Parameters
c An Android context for creating components
position Position of the Component that it's required.
Returns
  • The Component at the given position

public abstract boolean isAsyncLayoutEnabled ()

Depending on the returning value, this binder will try or not to measure the components in a background thread.

Returns
  • True to enable Async Layout, False otherwise.

public abstract boolean isIncrementalMountEnabled ()

Whether or not incremental should be enabled on components created by this Binder.

public abstract void notifyDataSetChanged ()

Notifies this Binder that the data has changed and all the ComponentTree need to be regenerated. This is very inefficient and the more granular notify methods should be preferred instead.

public abstract void notifyItemChanged (int position)

Notifies this Binder that the ComponentTree representing the item with index position needs to be regenerated as the data changed.

public abstract void notifyItemInserted (int position)

Notifies this Binder that an item was inserted at index position. If an item had index position before notifyItemInserted was called it is assumed that it will have index position + 1 after calling this method.

public abstract void notifyItemMoved (int fromPosition, int toPosition)

Notifies this Binder that an item moved from index fromPosition to index toPosition.

public abstract void notifyItemRangeChanged (int positionStart, int itemCount)

Notifies this Binder that the ComponentTrees representing the items with index starting at position and ending at position + itemCount need to be regenerated as the data changed.

public abstract void notifyItemRangeInserted (int positionStart, int itemCount)

Notifies this Binder that a range of items were inserted at index position. If an item had index position before notifyItemRangeInserted was called it is assumed that it will have index position + itemCount after calling this method.

public abstract void notifyItemRangeRemoved (int positionStart, int itemCount)

Notifies this Binder that a range of items were removed at index position. The items at index position + itemCount + i (with i >= 0) before notifyRangeRemoved was called will have index position + i after calling this method.

public abstract void notifyItemRemoved (int position)

Notifies this Binder that an item was removed from index position. The items at index position + i (with i >= 1) before notifyItemRemoved was called will have index position + i - 1 after calling this method.

public abstract void onBind (V view)
