Class Overview

Defines the relationship of a set of input values to a set of output values where the values from the input nodes 'flow into' the output nodes. For example, input values could be a touch X/Y or a layout value, and output values could be the X/Y position of a View or its opacity. Input and output values can be connected to each other via intermediate operators like springs or timing. NB: ValueNodes can be referenced by multiple GraphBindings (e.g. a view property).
